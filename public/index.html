Amazon CloudOps Security Workshop

Add yourself to the class roster and update your progress as you go through the workshop!

In this workshop, we will walk through the fundamentals of AWS security as a part of the AWS Cloud Operations course. By the end of this two-session module, you will be able to perform basic Linux and AWS administration tasks as a preparation for the following AWS Cloud Administrator bootcamp. Depending on your level of experience, you may want to have a look at the Further Reading section to learn more about Linux administration.

 


We’ll have instructor-led sessions to make sure everyone can to get through the basics - Exercises 1 through 5.
